Μicro Δynamic Κalculator

–•– About it –•–

As said in its GitHub README.md file, MDK is an experimental CUI function plotting application with no current pourpose or application.
Its interface asks for coefficients in a standard function which is going to be plotted, limiting its plotting capacity, but also leaving many possibilities.

It was the first Science52101 project to be made and the source of our logo, whose generation proccess' last inputted integer was '52101', the divisor for the last plotted function.
