Simple Object Reader

–•– About it –•–

As said in its GitHub README.md file, SOR is an experimental Simple Object Reader X File (.sorxf) file reader with Simple Object Syntax (SOS) support collection with no current pourpose or application.
It is a text evaluation and manipulation study and also a programming language creation experiment with the SOS (Simple Object Syntax) standard.

Akai Keisanki did many text evaluation studies with experimental programming-language-alike standards, but they were mostly quite simple and permanently useless.
